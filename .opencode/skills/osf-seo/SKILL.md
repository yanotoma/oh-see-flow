# SEO Specialist

Use this skill when optimizing for search engines, implementing meta tags, structured data, or improving discoverability.

## Best Practices

### On-Page SEO
- Unique, descriptive title tags (50-60 chars)
- Meta descriptions that encourage clicks (150-160 chars)
- One H1 per page, logical heading hierarchy (H1 → H2 → H3)
- Semantic HTML elements (article, nav, main, section)
- descriptive URLs (short, hyphened, lowercase)
- Internal linking with descriptive anchor text

### Technical SEO
- Fast page load (Core Web Vitals: LCP, FID, CLS)
- Mobile-first responsive design
- Canonical URLs to prevent duplicate content
- XML sitemap submitted to search engines
- robots.txt properly configured
- HTTPS everywhere
- Clean URL structure (no query params for content)

### Structured Data (JSON-LD)
- Organization/LocalBusiness for brand info
- BreadcrumbList for navigation
- Article/BlogPosting for content
- Product for e-commerce
- FAQ for common questions
- HowTo for tutorials

### Content Optimization
- Keyword research before writing
- Natural keyword placement (title, H1, first paragraph, alt text)
- Long-tail keywords for specific intent
- Content depth and comprehensiveness
- Regular updates to keep content fresh

### Images
- Descriptive file names (not IMG_001.jpg)
- Alt text that describes the image content
- Compressed images (WebP preferred)
- Lazy loading for below-the-fold images
- Proper sizing (don't serve 4K images for thumbnails)

### Links
- Internal links between related content
- External links to authoritative sources
- No broken links (404s)
- Descriptive anchor text (not "click here")
- Nofollow for untrusted/paid links

## Anti-patterns
- Keyword stuffing
- Hidden text or links
- Cloaking (different content for users vs crawlers)
- Duplicate content across pages
- Thin content (pages with little value)
- Ignoring mobile experience
- Missing alt text on images
- No sitemap
- Slow page loads
